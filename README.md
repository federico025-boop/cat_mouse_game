JUEGO DEL GATO Y EL RATÓN

 ¿QUÉ CREÉ?

Un juego de estrategia por turnos en un tablero 8x8 donde:

- El gato intenta atrapar al ratón
- El ratón intenta llegar al queso.
- Un jugador humano puede elegir ser gato o ratón ( g o r)
- La PC/IA juega con algoritmo Minimax.





Descripción general:
--------------------
Este proyecto implementa un juego presentando un tablero bidimensional (8x8) donde el gato y el ratón se mueven en las direcciones arriba,abajo,izquierda y derecha. El objetivo del gato es atrapar al ratón, por otra para el ratón tendra que cazar el queso para ganar!


Características Técnicas:
-------------------------
✅ Tablero dinámico con posición aleatoria del queso
✅ Movimientos validados (w/a/s/d)
✅ IA/PC con heurísticas diferenciadas según el rol
✅ Detección automática de victoria
✅ Alternancia de turnos automática




Estructura del código:
----------------------
- El código está organizado a traves de funciones usando el lenguaje Python
- Hay funciones para mostrar el tablero, generar los movimientos, verificar el fin del juego y ejecutar el algoritmo minimax.
- El archivo puede ejecutarse directamente y comenzará el juego en la consola.


Que fue el mejor AJA!:
--------------------
-Sin duda hacer uso del recurso de recursividad dentro del algoritmo minimax
-Tambien la asignación/interpretación de la heurística para que cada role cumpla con su objetivo.


¿Que fue un desastre:?
----------------------
- La implementación funcional entre el algoritmo minimax y la heurística de los personajes dependiendo de sus respectivos roles.
